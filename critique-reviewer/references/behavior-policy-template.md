## 8.6 상태·예외 동작 정책 (필수 디자인 6종)

1. **Loading** — 스켈레톤 + 진행률
2. **Empty (필터 결과 없음)** — 일러스트 + 완화 CTA
3. **Empty (데이터 자체 없음, 신규 환경)** — 일러스트 + 시작 안내 CTA
4. **Partial (일부 결측)** — 차트 빗금 + 상단 안내
5. **Error (수집기/API 장애)** — 에러 코드 + 재시도 + 운영팀 알림
6. **Forbidden** — 권한 없음 안내 + 요청 채널
