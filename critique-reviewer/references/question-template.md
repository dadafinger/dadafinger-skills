# 보강 질문지 A~L 섹션 템플릿

각 질문 끝에 `▶ 답:` 칸을 둔다. 결정 후 결정 로그로 이관.

우선순위:
- **P0 = 출시 블로커** — 답변 D+10 회수
- **P1 = 출시 전 필수** — 답변 D+31 회수
